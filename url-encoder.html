什么是 URL 编码？

URL 编码（也叫百分号编码）是一种用于将特殊字符转换为可以在 URL 中安全传输的格式的编码方式。例如，中文字符、空格、& 符号等都需要经过 URL 编码才能正确传输。

常见的 URL 编码示例：空格变为 %20，中文字符变为 %xx%xx 的形式。URL 解码则是将这些编码后的字符还原回原始内容。

本工具支持完整的 URL 编码和解码操作，包括 Unicode 字符的处理。所有操作均在浏览器本地完成，您的数据不会上传到服务器。
