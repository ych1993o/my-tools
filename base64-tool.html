什么是 Base64 编码？

Base64 是一种将二进制数据转换为 ASCII 字符串的编码方式，常用于在文本协议中安全地传输二进制数据。它将每 3 个字节（24 位）的数据转换为 4 个 ASCII 字符，编码后的数据体积会增大约 33%。

Base64 在实际开发中有广泛的应用场景：电子邮件附件的 MIME 编码、HTML/CSS 中的小图片内嵌（Data URI）、API 接口中的认证信息传递（Basic Auth）、JWT Token 的载荷部分等。

本工具支持包含中文字符的 Base64 编码和解码，采用 UTF-8 编码处理，能够正确转换中文、日文、韩文等 Unicode 字符。这是很多在线工具不支持的功能。

常见使用场景

  • 解码 API 响应中的 Base64 编码数据
  • 生成 HTML 中内嵌图片的 Data URI
  • 查看或修改 JWT Token 中的内容
  • 处理配置文件中的 Base64 编码字符串
  • 调试网络请求中编码的参数

所有编解码操作均在浏览器本地完成，您的数据不会发送到任何服务器。
